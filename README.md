# Project Pyper

![Pyper Logo](https://cdn.discordapp.com/attachments/902668705643782196/911644842147336292/rect938.png)

A new python compiler to statically convert python source code to executable files without any significant additions to the syntax of the language

## About

Pyper stands for **Python For Performance**. <br><br>
This project aims to statically compile python source code into machine-executable code without any major additions to the language's syntax.

Our goal is to make it possible to use Python to develop general mobile/desktop/web development. Without sacrificing performance for the sake of convenience. (Zero Cost Abstraction)
<br><br>
We hope it will be possible to use Python in performance-intensive projects one day.

## Golden Rules of Pyper
- Make **common case** fast.
- Avoid unnecessary **runtime** checks.
- Avoid unnecessary **complicated calculations**.
- Avoid **complicated data structures**; the simpler, the better.
- Stay as **_close_** to the **processor** and the **hardware** as possible.
- Let the programmers do what they love to do, but compile it down to do what the computers love to do.
